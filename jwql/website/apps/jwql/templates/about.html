About the project

The JWST Quicklook Application (JWQL) is a database-driven web application and automation framework for use by the JWST instrument teams to monitor the health and stability of the JWST instruments. Developed in Python, it exists within a private GitHub repository. The team actively strives to adhere to software engineering best practices, strictly following defined workflows and Python style guides.
